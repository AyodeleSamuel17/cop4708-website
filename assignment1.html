Overview

This assignment analyzes functional dependencies in a course enrollment dataset. Each potential dependency was evaluated and classified as valid, invalid, or partially valid based on the data.

Results

The table below summarizes the dependencies and their validity:

Dependency Validity Reason
StudentID → StudentNameValidEach student ID maps to exactly one student name.
StudentID → MajorValidEach student maintains the same major throughout the dataset.
StudentID → GPAInvalidStudents have varying GPAs across semesters.
CourseCode → CourseNameValidEach course code identifies a unique course name.
CourseCode → CreditsValidCredit hours are consistent per course.
InstructorID → InstructorNameValidEach instructor ID corresponds to one name.
CourseCode → InstructorIDPartially ValidCS101 is taught by multiple instructors.
RoomNumber → ClassSizeInvalidRoom capacities vary across courses.
(StudentID, CourseCode) → GradeValidEach student–course pair appears once.
(CourseCode, Semester) → InstructorIDValidEach course–semester combination has one instructor.
(CourseCode, Semester) → ClassSizeValidClass sizes are consistent per course-semester.
Major → GPAInvalidStudents within the same major have different GPAs.
InstructorName → InstructorIDValidNames uniquely map to instructor IDs.
